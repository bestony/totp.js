# TOTP.JS
This Script is a Time-based One-time Password Algorithm Wirtten By JavaScript.

I use it in Mini Program of WeChat.

Thanks  for project  @russau

### Usage
```javascript
const totp = require('../../utils/totp.js');
var token = totp.getCode(key)
console.log(token)
```
